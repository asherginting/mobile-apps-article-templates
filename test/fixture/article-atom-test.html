Related content now available in the Content API

We are excited to announce the first of many forthcoming changes to the Open Platform Content API

Michael Brunton-Spall & Stephen Wells

Published: 16:20 BST Tuesday, 17 August 2010

We have added the ability to find related content for a specific content item, in much the same way as our website shows you at the bottom of an article. We have also made some minor infrastructure changes behind the scenes that improves performance.

To get the related content, make a request to the item endpoint at http://content.guardianapis.com/open-platform/getting-started.json
and then add the show-related parameter, for example: http://content.guardianapis.com/open-platform/getting-started.json?show-related=true

This will return back the same results as before, but there will also be a relatedContent element list that contains the related items. The show-fields and show-tags parameters will apply to the related items as well as the main item, which should let you see the body, headline etc. of each item of related content.

We’ve updated the content API explorer to add a checkbox on the item endpoint to turn related content on and off. You can see this in the explorer

We hope you enjoy this feature and look forward to releasing more features soon. If you have a requirement or suggestion then we’d love to hear from you, feel free to contact us at openplatform@guardian.co.uk or via this mailing list.
